# Contributing to Construction-AI-Toolkit

Thank you for your interest in contributing to Construction-AI-Toolkit!

Our goal is to build the world's most comprehensive open-source collection of AI prompts, workflows, and engineering resources for the Architecture, Engineering, and Construction (AEC) industry.

We welcome contributions from engineers, BIM professionals, planners, estimators, project managers, quantity surveyors, HSE specialists, AI enthusiasts, and software developers.

---

# How You Can Contribute

You can contribute by:

- Improving existing prompts
- Adding new engineering workflows
- Fixing documentation
- Correcting technical errors
- Adding examples
- Improving prompt structure
- Translating documentation
- Reporting issues
- Suggesting new AI workflows

---

# Before You Start

Please:

- Search existing Issues before opening a new one.
- Discuss major changes before starting implementation.
- Keep contributions focused and well documented.
- Follow engineering best practices.
- Ensure technical accuracy.

---

# Pull Request Process

1. Fork the repository.
2. Create a feature branch.
3. Make your changes.
4. Test your work.
5. Update documentation if necessary.
6. Submit a Pull Request with a clear description.

---

# Prompt Contribution Guidelines

Every new prompt should include:

- Purpose
- AI Role
- Required Inputs
- Tasks
- Deliverables
- Output Format
- Best Practices

Use professional engineering terminology and internationally recognized standards whenever possible.

---

# Coding Standards

If contributing code:

- Follow clean coding principles.
- Include comments where necessary.
- Keep modules focused.
- Write reusable code.
- Avoid unnecessary dependencies.

---

# Documentation Standards

Documentation should be:

- Clear
- Professional
- Well structured
- Grammatically correct
- Easy to follow

Prefer Markdown formatting.

---

# Engineering Standards

Where applicable, reference recognized standards such as:

- ISO 19650
- PMI PMBOK
- AACE International
- FIDIC
- NEC
- RICS
- ASTM
- ACI
- AISC
- BS
- Eurocodes
- NFPA
- OSHA

---

# Reporting Issues

When reporting an issue, include:

- Description
- Expected behavior
- Actual behavior
- Steps to reproduce
- Suggested solution (if available)

---

# Code of Conduct

Please follow our Code of Conduct to ensure a respectful and collaborative environment.

---

# Thank You

Your contribution helps make AI more useful for the global engineering and construction community.

We appreciate every contribution, regardless of its size.
