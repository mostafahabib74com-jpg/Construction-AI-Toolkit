# 📑 Tendering Prompt Library

This section contains practical AI prompts for construction tendering, bid management, estimation, and technical proposal preparation.

## 🎯 Purpose

The goal of this library is to help construction professionals:

- Review tender documents efficiently.
- Identify missing requirements and risks.
- Prepare technical and commercial proposals.
- Analyze BOQs and pricing documents.
- Draft tender clarifications and RFIs.
- Develop execution methodologies.
- Improve bid compliance and submission quality.

---

## 📂 Planned Prompt Categories

- Tender Document Review
- Compliance Matrix
- Technical Proposal Preparation
- Commercial Proposal Preparation
- BOQ Analysis
- Pricing and Estimation
- Risk Identification
- Clarification and RFI Drafting
- Method Statement Preparation
- Project Schedule Development
- Resource Planning
- Bid Submission Checklist

---

## 🧠 Recommended Use

For best results, provide the AI assistant with:

1. Project name and location.
2. Client and consultant details.
3. Scope of work.
4. Tender documents.
5. BOQ and specifications.
6. Submission deadline.
7. Required technical and commercial deliverables.
8. Any exclusions, assumptions, or constraints.
