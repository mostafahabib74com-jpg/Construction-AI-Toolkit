# Code of Conduct

## Our Pledge

We are committed to creating a welcoming, respectful, and inclusive environment for everyone, regardless of background, experience, identity, or skill level.

## Our Standards

Examples of positive behavior include:

- Being respectful and professional
- Accepting constructive feedback
- Helping others learn
- Sharing knowledge openly
- Using inclusive language
- Focusing on technical discussions

Examples of unacceptable behavior include:

- Harassment
- Personal attacks
- Discrimination
- Trolling
- Offensive language
- Publishing private information

## Enforcement

Project maintainers are responsible for enforcing this Code of Conduct and may remove comments, issues, or contributions that violate these guidelines.

## Scope

This Code of Conduct applies to all project spaces, including GitHub Issues, Pull Requests, Discussions, and community interactions.

## Contact

If you experience unacceptable behavior, please open a confidential issue or contact the project maintainer.

Thank you for helping build a professional engineering community.
