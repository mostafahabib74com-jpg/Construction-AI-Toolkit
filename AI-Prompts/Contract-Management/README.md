# 📑 Contract Management Prompt Library

## Overview

This library provides professional AI prompts for managing construction contracts throughout the project lifecycle.

It supports Owners, Consultants, Contractors, Contract Administrators, Claims Engineers, Planning Engineers, Commercial Managers, and Project Managers.

The prompts cover contract administration, claims preparation, extension of time (EOT), variation orders, payment applications, contractual correspondence, dispute resolution, final accounts, and FIDIC-based project management.
