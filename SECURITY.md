# Security Policy

## Reporting a Vulnerability

If you discover a security issue, please do not create a public GitHub issue.

Instead, contact the project maintainer privately with:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested mitigation (if available)

We will investigate and respond as quickly as possible.

## Supported Versions

Currently, the latest version of this repository is supported.

Thank you for helping keep the project secure.
